Background: parallax

Parallax effect build with jquery.stellar plugin.

You can easily add or remove a parallax effect for all sections. Just add data-stellar-background-ratio  or remove it.

HTML

<header id="header" class="header masked" data-stellar-background-ratio="0.6">
